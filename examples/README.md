# 使用场景示例

## 场景一：换电脑后恢复团队

### Step 1：旧电脑上打包

用户在旧电脑上对 Agent 说：

> 我要搬家

Agent 执行 `pack.py`，输出：

```
检测到团队成员：
  - xiaoce（小策）
  - laomo（老墨）
  - dabai（大白）
  - ...

打包完成：~/Agent搬家包_20260422.zip
请将压缩包复制到新环境
```

### Step 2：新电脑上迁移

用户把压缩包放到新电脑，对 Agent 说：

> 这是搬家包，帮我搬家

Agent 执行 `migrate.py`，按提示引导用户：
- 解压文件
- 选择 Main Agent 处理方式
- 选择子代理处理方式
- 配置合并（不覆盖已有）

### Step 3：验证

迁移完成后，验证：
- Agent 能正常唤醒
- cron 任务已恢复
- 成员配置正确
